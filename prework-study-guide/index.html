HTML

  • HTML is short for Hypertext Markup Language.
  • The !DOCTYPE declares to the browser what document type to expect.
  • Two of the most important elements of HTML are as follws:
    1. The head
    2. The body
  • The head element contains information about the webpage.
    • Elements located in the head are not visible to the user looking at the web page.
    • The information located in the head is called metadata
  • The body element represents the visible content shown to the user.
    • The visible content of the webpage is located in the body.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git init: initialises an existing directory as a Git repository
  • git clone [url]: retrieves an entire repository from a hosted location via URL
  • git commit: saves the changes in your local repository
  • git push: pushes the committed file changes from the local repository to the remote repository
  • git pull: fetches the last uploaded changes from the remote server into the local repository
  • git merge: merges your branch with the parent branch
